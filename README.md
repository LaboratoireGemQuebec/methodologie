# Laboratoire Gem Quebec - Open Methodology

This repository documents the analytical methodology used at Laboratoire Gem Quebec (LGQ), Laval, Quebec, Canada, for gemmological identification, geographic origin determination, treatment detection, and certification of coloured stones, diamonds, and organic gem materials.

## Mission

Provide reproducible, open, and citable methodology to the gemmological research community, in the spirit of scientific transparency advocated by the Laboratory Manual Harmonisation Committee (LMHC) and CIBJO Blue Books.

## Multi-instrument analytical platform

| Instrument | Manufacturer / Model | Primary use |
|---|---|---|
| FTIR | Thermo Scientific Nicolet iS5 | Treatment detection, hydrocarbon characterisation, polymer detection |
| UV-Vis-NIR | Magilabs GemmoSphere | Color cause analysis, treatment identification, origin clues |
| Raman | Renishaw | Mineralogical identification, inclusion analysis, polymorph discrimination |
| EDXRF | Bruker Tracer 5i | Elemental composition, geographic origin clues, treatment chemistry |
| Microscope | Nikon SMZ1270 | Inclusion observation, growth pattern analysis |
| Refractometer | Eickhorst LED model | Refractive index, birefringence, optic character |
| Polariscope | GIA GemInstruments | Optic figure, anomalous double refraction |
| Gamma spectrometer | Ortec DigiBASE | Radioactivity screening (Bq/g) for irradiated stones |
| Analytical balance | Mettler Toledo NewClassic MS | Mass and specific gravity |
| UV cabinets | 365 nm long-wave + 254 nm short-wave | Fluorescence patterns |

## Standard analytical workflow

1. Visual inspection: color, transparency, lustre, surface features, photograph documentation
2. 2. Mass measurement: analytical balance to 0.001 g
   3. 3. Dimensional measurement: digital calliper or microscope reticle
      4. 4. Refractive index: Eickhorst LED refractometer
         5. 5. Specific gravity: hydrostatic weighing if RI inconclusive
            6. 6. Optic character: polariscope examination
               7. 7. Spectroscopy short-cycle: UV-Vis-NIR for color cause and treatment screening
                  8. 8. Microscopy: 10x to 80x for inclusion identification, treatment evidence
                     9. 9. Decision tree: escalate to step 10 if identification ambiguous
                        10. 10. Spectroscopy long-cycle: FTIR, Raman, EDXRF
                            11. 11. Radioactivity screening: gamma spectrometer if irradiation suspected
                                12. 12. Cross-reference: LMHC, CIBJO Blue Books, Hughes / Pardieu / Gubelin / SSEF / GIA
